NoNameIdee est un bot discord utilisant discord.py pour gérer un serveur discord universitaire.

Il ne possède que trois fichier qu'il suffit s'implement de télécharger.
BotPython.js et le code entier du bot.
Les fichier sauv.ods et config.json sont des bases de données utilisés si le bot doit être éteint puis rallumé.

Avant utilisation il faut vérifier que tout les modules utilisés sont bien importé par la machine (normalement il n'y a besin de faire qu'un: pip install discord et un pip install pyexcel_ods)
Pour le faire fonctionner il suffit de démarer BotPython.py et le bot sera opérationnel tant que la console n'ai pas fermé.
Il est sinon possible de le mettre en ligne sur un serveur via un site contre un abonnement pour qu'il ne s'éteingne pas.

Une fois le bot opérationnel la commande !help donne la liste et une rapide explication des différentes commandes que peut utiliser la personne.
Seul les commandes que peut faire l'utilisateur de la commande (suivant ses rôles) vont être affiché.
